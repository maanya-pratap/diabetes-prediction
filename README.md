# Diabetes Prediction using Logistic Regression

## Description
This project predicts whether a patient is diabetic using the Pima Indians Diabetes Dataset and the Logistic Regression algorithm.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Dataset
Pima Indians Diabetes Dataset

## Output
- Data preprocessing
- Model training
- Accuracy score
- Confusion Matrix
- Classification Report
- Prediction for a new patient
